📝 연습문제
1. 기본 집계
groupby를 이용해서 학년별 국어 평균 점수를 구하라.
pivot_table을 이용해서 학년별, 반별 영어 평균과 수학 평균을 구하라.

2. 고급 집계
agg를 이용해서 각 학년별 국어 평균, 영어 최고점, 수학 표준편차를 구하라.
apply를 이용해서 학생별 (국어+영어+수학+과학)/4 평균을 직접 계산해서 평균 컬럼을 추가하라.

3. 데이터 합치기
merge를 이용해서 students와 clubs 데이터를 합쳐라.
합친 데이터에서 동아리별 출석일수 평균과 과학 점수 평균을 구하라.

4. 구간 나누기 (cut)
전체 평균 점수(평균)를 기준으로 학점을 나누어라.
90 이상 = A
80 이상 = B
70 이상 = C
60 이상 = D
60 미만 = F
(pd.cut 활용)

5. 응용 문제
학년별, 동아리별로 국어 평균과 영어 평균을 pivot_table로 만들어라.
apply를 이용해서 출석일수가 15일 이상이면 "우수", 아니면 "보통"이라는 새로운 컬럼(출석등급)을 만들어라.
모든 과목 평균이 85 이상인 학생만 뽑아서 새로운 DataFrame을 만들어라.
